This is a project by Vivek in which I implemented a Spotify recommendation system that takes a playlist link as an input and outputs a list of song recommendations based on various audio features.

Steps to get a list of song recommendations:
1.Copy a playlist link from spotify app.
2.Paste it in the input field.
3.Select the number of recommendations you want.
4.Click on the get recommendations button.

You can play any song by just clicking on the play button or the track name.
